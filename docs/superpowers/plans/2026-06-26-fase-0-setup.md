# Fase 0 — Setup del Monorepo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Tener un monorepo funcional con backend Go (Gin) + frontend SvelteKit + Postgres local + Docker Compose, donde `make dev` levanta todo y `curl /health` + `curl localhost:5173` responden.

**Architecture:** Monorepo simple con `backend/` (Go) y `web/` (SvelteKit), coordinados por `docker-compose.yml` para Postgres. Cada subsistema tiene su propio `Makefile` con atajos. Backend expone REST en `:8080`, frontend Vite dev server en `:5173`. Comunicación frontend↔backend via HTTP/JSON, CORS abierto en dev.

**Tech Stack:**
- Go 1.23+, Gin v1.10+, GORM v2 (preparado, no usado aún)
- PostgreSQL 16 (via Docker Compose)
- SvelteKit 2 + Svelte 5 (runes), TypeScript estricto, Vite, Tailwind CSS v4
- pnpm 9+ workspaces
- golang-migrate para migraciones (preparado, no usado aún)

## Global Constraints

- **Go:** versión 1.23 o superior (verificar con `go version`)
- **Node:** versión 20 LTS o superior (verificar con `node --version`)
- **pnpm:** versión 9 o superior (`npm install -g pnpm` si falta)

```go
package config

import (
	"log"
	"os"

	"github.com/joho/godotenv"
)

type Config struct {
	Port        string
	DatabaseURL string
	JWTSecret   string
	GinMode     string
}

func Load() *Config {
	if err := godotenv.Load(); err != nil {
		log.Println("No .env file found, using system env")
	}

	return &Config{
		Port:        getEnv("PORT", "8080"),
		DatabaseURL: getEnv("DATABASE_URL", ""),
		JWTSecret:   getEnv("JWT_SECRET", ""),
		GinMode:     getEnv("GIN_MODE", "debug"),
	}
}

func getEnv(key, fallback string) string {
	if v := os.Getenv(key); v != "" {
		return v
	}
	return fallback
}
```

- [ ] **Step 1: Crear `internal/db/db.go`**

```go
package db

import (
	"log"

	"gorm.io/driver/postgres"
	"gorm.io/gorm"

	"github.com/nicolas/finanzas/backend/internal/config"
)

func Connect(cfg *config.Config) (*gorm.DB, error) {
	if cfg.DatabaseURL == "" {
		log.Fatal("DATABASE_URL is empty")
	}

	gormDB, err := gorm.Open(postgres.Open(cfg.DatabaseURL), &gorm.Config{})
	if err != nil {
		return nil, err
	}

	sqlDB, err := gormDB.DB()
	if err != nil {
		return nil, err
	}

	if err := sqlDB.Ping(); err != nil {
		return nil, err
	}

	log.Println("Database connection established")
	return gormDB, nil
}
```
